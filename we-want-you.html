We Want You

Post in 三 29 三月 2017 |Tags Python Geology Kivy Group

你还在单打独斗么？加入我们！

图片来自网络，侵删。

0. 起因

2016 年底，我经历了一次肿瘤手术。

进入手术台之前，我觉得一旦自己出不来，那就太郁闷了，因为我有好多想做的事情都没有做，要么没动手尝试，要么没有做完或者没有做好。

从手术台出来之后，感觉自己好像是用最后一枚筹码赢回了重新活下去的机会。所以这一次我要把所有想做的事都努力做好。

1. Kivy 中文开发者群组

继之前翻译的 Think Python 第二版 之后，我又翻译了 Kivy中文编程指南。一来是多学习，二来是把自己的经验分享给其他可能跟我一样刚入门而又需要更多资源的初学者们。

然而我一个人的力量终究很微弱，所以后来先后有了@曹瀚 Robert 、面包 以及 小涂 等朋友的加入，其中@曹瀚 Robert 在开始翻译 Kivy 的 API 文档， 茶杯 创立了最早的 Kivy 中文社区，小涂 则参与了在 Kivy中文编程指南 中的图件汉化。

我还加入了由 桂老师 创建的群 248136053，以及由 面包 的群 310972838，遇到了更多的 Kivy 中文开发者。

然而我发现大家还是都觉得很少有中文的 Kivy 项目，更少有本土化的 Kivy 应用。所以我想，大家不妨从较为封闭的网络论坛、QQ 群，拓展到更宽泛更开放的平台。

于是我就在 Github 上面建立了Kivy Developers From China 这个群组，并把自己之前翻译的 Kivy 中文开发指南 迁移过去，这样大家就可以共同来对它进行改进、更新和完善。然后我又建了一个 Kivy 探索之路项目，是希望老手们可以分享一下自己踩过的坑，等等。

当然，目前都还没有什么内容。

所以，如果你对 Kivy 感兴趣，如果你是 Kivy 的中文开发者，希望你能加入Kivy Developers From China 群组。可以将你的 Github ID 私信或者留言给我，然后我去发出邀请。

2. 地质学家群组

这个小标题我吹牛了，抱歉，我实在算不上地质学家，虽然某些国际上顶级的老前辈称呼我为 Young Geologist。我自认为自己顶多也就是个地质学工作者，我这里用是因为加入到 China Geology 项目的朋友中有很多都是优秀的地质学家。

2015 年左右，我甚至考虑把手上的 http://ChinaGeology.com 域名卖出去，联系了 中国地质 这家杂志社，但是对方根本没理睬我。情理之中。

后来，我还联系过一家专注地学领域的翻译公司的创始人，他似乎对 http://ChinaGeology.com 这个域名有点兴趣，但更希望我能参与到建设和运营中。我当时就拒绝了。

因为当时我只是需要一笔钱来交学费，后来没办法我就把当时在开发的一个移动应用雏形整个卖掉了，交了学费。之后我的身体出了各种状况，直到 2016 年末手术。

嗯，跟开头续上了，除了翻译了 Think Python 第二版 和 Kivy 中文开发指南 ，我还在 2017 年初，把自己用 Python 来完成一些地质学日常任务的脚本积攒了起来，打包成了一个 GeoPython 地质学开源工具集，分享给遇到类似情形的地学同行。

其实都是一些很简单的功能，基于 Excel 表格或者 CSV 文本生成通用矢量图格式 SVG 文件，能直接在 Adobe Illustrator、CorelDRAW X4 等软件内编辑。

GeoPython 有三个主要特点，开源，免费，跨平台。

开源，所有源代码公开，基于 GPL 协议。免费，随便用，我是不会收钱的。跨平台，其实很简单，因为是 Python 下的模块，可以 pip install geopython 安装。所以只要能运行 Python 的平台就都可以用。普遍的 Windows 用户可以用，我这种 Linux、Mac 用户可以用，甚至 BSD 系统用户都能使用。

若干 功能都是其他软件已经实现过的，并且可能做得比眼下的 GeoPython 还要更完善。那么为什么还要 重新制造一个轮子**呢？首先，因为其他软件好像都不同时具备开源、免费、跨平台这三个特点；另外，其他各个学科都已经有了大量的基于 Python 产生的专业工具库，而我觉得地质学上面的还很有扩展空间。

我的力量依然还是那么薄弱，无论是编程开发的能力，还是地学领域的学术造诣，都很低端。

我想我还是需要团结大家的力量，于是我在 Github 上面创建了一个 Geologists from China 群组。

这个 Geologists from China 群组是做什么的呢？其实也无非就是分享和交流，只不过更多地以代码的形式，共同开发项目、分享自己的工具、参与开源工具的维护等等。目前里面包括了我创建的 GeoPython ，以及 斯坦森.李 创建的 Mineral-Formula-Calculation ，还有就是基于 Github Pages 和 Pelican 搭建的 ChinaGeology.com。

是的，这次我不打算把域名卖点钱拿去交学费了，这主要感谢我夫人对我的赞助让我免去了学费的烦恼。我曾经一度希望能把 ChinaGeology.com 搭建成一个地学科普平台，或者一个开源的在线出版平台之类的。

所以，如果你对地质学感兴趣，如果你是中国的地学工作者，希望你能加入Geologists from China 群组。可以将你的 Github ID 私信或者留言给我，然后我去发出邀请。

总结

你可能会发现，我似乎很喜欢 开源 这个词。

是的，我是开源软件的忠实信徒，分享和帮助他人是我的爱好，我希望在自己有限的生命中，尽最大可能为大家提供有用的东西，哪怕只有一丁点用处，也是好的。

这个时代充满信息洪流，单个人能承载的知识、能力，再多也都显得微不足道。

我一个人黯淡苍白的，如果我们的光芒聚集到一起就可能照亮无限远的地方。

请将你的 Github ID 私信或者留言给我，当然也要注明你要加入的是 Kivy Developers From China 还是 Geologists from China。
